Main Description
Um documento de design de jogo é um documento altamente descritivo e detalhado do design de um jogo. Um GDD é um documento ativo, o que significa que é propenso a feedback. Toda vez que os requisitos mudam, um GDD deve mudar. Geralmente, um GDD é criado e editado de forma colaborativa por desenvolvedores e designers e usado para organizar os esforços dentro da equipe.
