Main Description

Quando o teste é realizado, algumas equipes preferem trabalhar usando a metodologia tik-tok, alternando entre o desenvolvimento de novos recursos e a correção de bugs. Atrasar todas as correções de bugs é extremamente ineficaz e perigoso: reduz a eficácia dos testes e pode levar ao acúmulo de problemas difíceis de resolver.
